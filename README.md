# Loop Habit Tracker (Cross-Platform)

A cross-platform mobile app for creating and maintaining long-term positive habits, built for both Android and iOS.

This is an independent project based on [Loop Habit Tracker](https://github.com/iSoron/uhabits) by Alinson Santos Xavier, rewritten as a cross-platform application. It is not affiliated with or endorsed by the original project.

## Origin and Attribution

This project is derived from [Loop Habit Tracker (uhabits)](https://github.com/iSoron/uhabits), an open-source Android app originally created by **Alinson Santos Xavier** (`<isoron@gmail.com>`). The original app is an excellent habit-tracking tool for Android — this project aims to bring a similar experience to both Android and iOS through a cross-platform rewrite.

### What changed

This is a cross-platform rewrite, not a direct fork. The original project is a native Android app built with Kotlin/Java. This project reimplements the core concepts and functionality for both Android and iOS.

## Features

*Planned features inspired by the original Loop Habit Tracker:*

- Simple, clean interface following Material Design guidelines
- Habit score that measures how strong your habits are
- Flexible scheduling (daily or specific days of the week)
- Reminders with customizable notifications
- Detailed statistics and charts for each habit
- Data export (CSV)
- Ad-free, open source, and respects your privacy
- Works fully offline with no external services required
